## Introduction
TripMate is a personalized trip suggestion app that helps users discover hidden gems and plan their next adventure based on their interests. This project is part of a college initiative to explore various technologies and implement them in a cohesive application.

## Tech Stack
TripMate utilizes a diverse set of technologies to deliver a seamless experience:

- **Frontend:**
  - **HTML**: Structure the web pages.
  - **CSS**: Style the web pages.
  - **Tailwind CSS**: Utility-first CSS framework for styling.
  - **JavaScript**: Add interactivity to the web pages.
  - **EJS**: Embedded JavaScript templating.

- **Backend:**
  - **MERN Stack**: 
    - **MongoDB**: Database to store user data and trip information.
    - **Express.js**: Web application framework for Node.js.
    - **React.js**: Frontend library for building user interfaces.
    - **Node.js**: JavaScript runtime for server-side development.
  - **Java Spring Boot**: Used for specific backend services, providing a robust and scalable framework for enterprise applications.

## Features
- Personalized trip suggestions based on user interests.
- Discover hidden gems and local attractions.
- Plan trips and itineraries.
- User authentication and profile management.
